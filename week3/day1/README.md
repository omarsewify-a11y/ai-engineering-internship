# Week 3 - Day 1: Data Preparation & Model Evaluation (Iris Dataset)

## Objective
This task focuses on preparing data correctly and evaluating a machine learning model using multiple evaluation metrics.

## Dataset
Iris dataset (built-in dataset from scikit-learn)

## Tasks Completed
- Loaded dataset from sklearn
- Checked for missing values
- Encoded target labels (already numerical in dataset)
- Split data into training and testing sets
- Trained a Decision Tree Classifier
- Evaluated model using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Key Learning Outcomes
- Understanding structured datasets
- Train/test split importance
- Model evaluation using multiple metrics
- Interpretation of confusion matrix, precision, recall, and F1-score
